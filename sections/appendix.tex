\subsection{Ray-Tracing Post-Processing Metric Details}
\label{app:raytrace_metric_delta_details}

Section~\ref{sec:results_raytrace_postproc} reports the controlled ablation with and without physics-based post-processing using the seed-averaged paired heatmap and aggregate summary table. All distributions use the Patent+Synth-full TabSyn tuned run with diffusion learning rate $3\times10^{-4}$. Across the four sampling seeds, the branch without post-processing has 6{,}714 \ensuremath{\pm} 63 feasible rows and the branch with post-processing has 8{,}371 \ensuremath{\pm} 28 feasible rows out of the same 10{,}000 generated-row denominator. SPO and encircled-energy metrics are normalized by Airy diameter; nominal performance is left in its returned units.
